## 문제 설명
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

## 해결 과정

### 접근 방법
반복문을 통해 n까지 모든 수를 합하기

## 코드
```python
n = int(input())
a = 0
for i in range(n):
    a += n
    n -= 1
print(a)

# 반복문 부분을 a += i+1 로 해도 됐을 것 같다
# 아니면 1부터 n까지 합이니까 (n+1) * int(n/2) 이런 걸 이용하는 것도 괜찮았을 것 같다
